This project demonstrates a complete end-to-end Machine Learning workflow, starting from data loading and preprocessing to model training, evaluation, and predictions.
It is designed to help beginners and intermediate learners understand how ML works in real-world scenarios using a structured and practical example.

This project uses the dataset excalories.csv, which contains calorie-related data used to train a prediction model.
